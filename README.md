📈 Stock Price Forecasting using ARIMA (Time Series Analysis) 

📌 Project Overview

This project focuses on forecasting stock prices using historical market data and statistical time series modeling. The goal is to analyze past price behavior and build a predictive model using Auto ARIMA, with a recursive forecasting approach to simulate real-world prediction scenarios. The model is also used to generate a one-year ahead forecast, providing insights into potential future price trends and uncertainty ranges.

The project uses IBM stock data as a case study and evaluates model performance using standard regression metrics.
